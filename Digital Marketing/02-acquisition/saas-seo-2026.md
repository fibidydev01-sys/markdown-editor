## 🤖 OPTIMASI UNTUK AI OVERVIEW & LLM

### Apa yang Bikin Konten Lo Di-cite AI?

Per AirOps 2026 research (April):

**Content depth dan readability matter most.**

Format yang work:
1. **Comparison pages dengan 3+ tabel** → 25.7% more citations
2. **Validation pages dengan 8+ list sections** → 26.9% more citations
3. **Shortlist pages ≤10 kata per kalimat** → 18.8% more citations

**Translasi praktis:** Tulis dengan **struktur jelas + kalimat pendek + tabel/list**.

### Format yang AI Models Bisa Read

Penelitian OtterlyAI April 2026: **AI hanya cite HTML pages, ignore Markdown (.md) pages**.

**Implikasi:** Render konten sebagai HTML proper (paragraph tags, list tags, table tags), bukan markdown-converted text.

### Pertanyaan-Driven Content

Buyer 2026 ngga search "best CRM" lagi. Mereka ask:
- *"What's the best CRM for B2B SaaS with 10-50 employees that integrates with Slack?"*
- *"How does HubSpot compare to Pipedrive for early-stage startups?"*

**Content structure:**
- Headline = pertanyaan jelas
- First paragraph = direct answer (40-60 kata)
- Body = supporting detail, table, list

## 📊 SCHEMA MARKUP — WAJIB 2026

Schema markup = structured data yang ngebantu search engine + AI parse konten lo dengan jelas.

**Schema types prioritas SaaS:**

### 1. SoftwareApplication Schema
Untuk halaman produk lo:
```json
{
  "@type": "SoftwareApplication",
  "name": "Your Product",
  "applicationCategory": "BusinessApplication",
  "operatingSystem": "Web",
  "offers": {
    "@type": "Offer",
    "price": "29",
    "priceCurrency": "USD"
  },
  "aggregateRating": {...}
}
```

### 2. FAQPage Schema
Untuk FAQ section:
```json
{
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "What is X?",
    "acceptedAnswer": {"@type": "Answer", "text": "..."}
  }]
}
```

### 3. Organization Schema
Untuk brand-level entity recognition.

### 4. Product Schema (untuk feature pages)

**Tools:**
- Schema.org Markup Generator (free)
- Google Rich Results Test (validate)
- Yoast SEO / RankMath plugin (WordPress)

## 🔗 PILLAR YANG TETAP RELEVAN

### 1. Technical SEO Foundations
Tetap kritikal:
- **Core Web Vitals** (LCP, CLS, FID) — Google rank factor + AI crawl signal
- **Mobile-first** indexing
- **HTTPS, clean URL structure**
- **Sitemap** + robots.txt proper
- **Page speed <2 detik**

### 2. Backlink Quality > Quantity
AI models pay attention ke **domain authority** untuk decide trust.

Strategi 2026:
- **Digital PR** (jadi sumber data/research yang quotable)
- **HARO replacement** (Help A Reporter Out udah ditutup; pakai Connectively, Qwoted, SourceBottle)
- **Guest post di publication top niche lo**
- **Tools/data sebagai linkable asset**

### 3. Topical Authority
Cover topik secara comprehensive:
- 1 pillar page (broad topic)
- 10-30 cluster pages (subtopic, internal link ke pillar)
- Update regular = freshness signal

### 4. User Intent Match
Match content type ke intent:
- Informational query → guide/blog
- Comparison query → comparison page
- Transactional query → landing/product page

## 🎯 TOPICAL CLUSTERS — STRUKTUR YANG WORK

**Hub & Spoke Model:**

```
[Pillar Page]
   "Complete Guide to [Category]"
       /          |          \
[Cluster 1]   [Cluster 2]   [Cluster 3]
"Best X for Y" "X vs Y"     "How to X"
```

**Aturan:**
- Pillar page = 3000-5000 kata, broad coverage
- Cluster page = 1500-2500 kata, specific aspect
- Tiap cluster link **balik ke pillar**
- Pillar link **ke semua cluster**

**Why it works:**
- Signal Google + AI bahwa lo expert di topik ini
- User journey natural (pillar → spesifik)
- Internal link equity flow

## 💎 BRAND RECOMMENDATION RATE — METRIC BARU

Per Optimist 2026, metric yang harus lo track baru:

**Brand Recommendation Rate (BRR)** = % AI responses to relevant queries that mention or recommend your brand by name.

**Cara track:**
1. List 50 query relevan ke produk lo
2. Tiap minggu/bulan, run query di ChatGPT, Claude, Perplexity, Gemini, AI Overview Google
3. Hitung % yang mention brand lo
4. Track trend over time

**Tools yang membantu:**
- **Otterly.AI** — track AI search visibility
- **Profound** — AI brand monitoring
- **Athena HQ** — LLM visibility tracker
- **Manual** spreadsheet (gratis tapi tedious)

**Target:** Naik 5-10 percentage points dalam 6 bulan.
